## Filled TTFs — Google Fonts (all SIL OFL 1.1)
mr-de-haviland · pinyon-script · herr-von-muellerhoff · cinzel · cormorant ·
bodoni-moda · libre-caslon-display · vidaloka · luxurious-script · tangerine

These TTFs are used SERVER-SIDE ONLY (opentype.js text→path outlining at save
time) so the saved monogram SVG is self-contained pure paths. They are not
served to browsers; the editor preview loads the same faces from Google Fonts.
