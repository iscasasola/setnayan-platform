## 2026-06-23 · feat(event-type): per-profile role-set foundation — iteration 0053 Phase 2 (PR-1)

The foundation for per-event-type guest roles + seating tiers (spec `0053_event_type_engine`). Today the guest-role picker and the seating auto-fill tiers are hard-coded to the wedding role set; this introduces a per-profile `RoleSet` so a non-wedding event can offer a small generic role set and tier sensibly. **Zero behaviour change** — nothing resolves a non-wedding set yet, and weddings are byte-identical.
