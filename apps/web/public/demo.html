5 · Voice greeting (Web Speech API — built into your browser)

Hear your name spoken

Type any name and click play. Your browser will speak it using the built-in Web Speech API (free, no service needed). For production we'd upgrade to a higher-quality neural voice (ElevenLabs, Google Cloud TTS) — still ~$0.001 per guest. Demo uses the browser's voice for instant proof-of-concept.

Cost · ~$0.001 per guest with neural TTS / $0 with browser voice

Tip: Chrome and Safari have higher-quality voices than Firefox. On a real invitation we'd cache one MP3 per guest, generated once.
